Welcome to My Task Manager!

This to-do list application was created by me as a coding project to improve my JavaScript, HTML, and CSS skills while building something useful for everyday life.

Features include task management with due dates, filtering options, and data that persists in your browser's local storage.

Feel free to explore the features and use this tool to organize your tasks!

About This Project

This Task Manager was developed by me as part of my journey to improve my web development skills.

The application is built with vanilla JavaScript, HTML, and CSS without any external libraries or frameworks, demonstrating core web development concepts like DOM manipulation and local storage use.

The project helped me learn about:

  • Creating interactive web applications
  • Managing application state
  • Persisting data in the browser
  • Building responsive layouts
  • Implementing filter and sort functionalities

I'm continuously improving this project and welcome any feedback!
